# CV Repository

This repository contains my professional curriculum vitae and related job application resources. It's set up to be viewed both on GitHub and through GitHub Pages.

## GitHub Pages

This repository is configured to publish to GitHub Pages automatically. The CV can be viewed online at:
`https://teetow.com/cv/`

## Development

To update the CV:

1. Edit the Markdown files (cv.md, cv_long.md, etc.)
2. Run `python generate_index.py` to update the index page
3. Commit and push the changes
4. GitHub Actions will automatically deploy the site
